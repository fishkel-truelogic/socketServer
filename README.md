socketServer
============

## Un programita hecho en JavaScript para probar las implementaciones de las funciones de fereSocket.c

### Instrucciones

* sudo apt-get update
* sudo wget http://nodejs.org/dist/v0.10.32/node-v0.10.32.tar.gz
* tar -xvzf node-v0.10.32.tar.gz
* cd node-v0.10.32/
* ./configure
* make
